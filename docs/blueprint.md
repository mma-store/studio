# **App Name**: MMA - Official Platform

## Core Features:

- Multi-Role Role-Based Access Control: Secure authentication and permission system for Admin, Sales, Technicians, Warehouse, and Retail/Wholesale customers using Firebase Auth.
- Dynamic Spare Parts Catalog: A high-performance product browsing experience with categories, motorcycle model filters, and promotional banners fetched from Firestore.
- AI Compatibility Tool: A smart tool that uses generative AI to help customers and staff verify if a specific part is compatible with a particular motorcycle year/model.
- POS & Sales Management: A clean Point of Sale interface for sales employees to process retail and wholesale transactions quickly on any device.
- Workshop Task Management: A digital tracking system for technicians to manage maintenance workflows, status updates, and repair histories.
- Inventory & Warehouse Module: Real-time stock management and warehouse alerts ensuring high accuracy across thousands of SKU items.
- RTL Responsive PWA Layout: A production-ready application shell featuring right-to-left layout support, offline-first capabilities, and Material Design 3 components.

## Style Guidelines:

- Primary Color: High-vibrancy Orange (#FF6B00) symbolizing mechanical power and energy. Background: Sophisticated Off-white (#FDF8F5) with a slight warm undertone.
- Accent: Analogous Red-Orange (#D32F2F) for highlights and primary CTAs to ensure high contrast against charcoal text elements.
- Font Recommendation: 'Poppins' for headlines and body text to achieve a contemporary, precise, and avant-garde look. Note: currently only Google Fonts are supported.
- Material Design 3 with full RTL (Right-to-Left) orientation. 12px grid-based spacing, large touch targets, and bottom navigation for mobile-first utility.
- Industrial yet minimalist line icons following the Material Symbols library, specifically themed for motorcycle maintenance and mechanical tools.
- Smooth micro-interactions including skeletal loading states for inventory queries and a modern pull-to-refresh effect on dashboard views.
